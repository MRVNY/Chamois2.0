Dans la scène game les objets utilisés par le kit sont:
- UI
- NPC Collection
- StoryItem Collection
- UI Camera
- Controllers

Controllers est ce qui gère le fonctionnement quasi complet de l'asset

UI Camera se charge d'afficher les éléments de l'UI de l'asset, pour nous, les dialogues

UI est donc tous les éléments qui peuvent être afficher

StoryItem Collection est utilisé pour les systèmes de quêtes soit pour compléter soit en tant que reward

NPC Collection est la liste des PNJ, sur cet objet nous avons ajouter des surcouches et fait des modifications
